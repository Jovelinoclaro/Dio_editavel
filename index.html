Projeto DIO

Descrição do Projeto

O Projeto DIO é uma aplicação para cadastrar dados no sistema de Distribuidores Internos de Fibra Óptica, mais conhecidos como DIO.

Passos para Configuração

  1. Criação do banco de dados: Você pode criar o banco de dados manualmente ou com o arquivo Python disponível.
    • Localize o arquivo: bancoDados.py.
    • Execute o arquivo para criar a estrutura do banco de dados automaticamente.
  2. Estrutura SQL para a tabela: Caso prefira criar manualmente, use o comando SQL abaixo: CREATE TABLE IF NOT EXISTS `lab_01` ( `id` int NOT NULL AUTO_INCREMENT, `dio_fibra` int NOT NULL, `dio_a` varchar(220) CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci NOT NULL, `dio_b` varchar(220) CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci NOT NULL, `dio_c` varchar(220) CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci NOT NULL, PRIMARY KEY (`id`) ) ENGINE=MyISAM DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci;
  3. Editar dados: Ao clicar na opção Editar dentro da aplicação, os dados serão editados e salvos no mesmo local automaticamente.
